# 全粒粉100%食パンを焼いた

普段、お昼はサンドイッチを食べてます。
週末に1週間分の食パンを焼いて冷凍保存しておき、朝は具材を挟むだけなのでお手軽です。

ちょっと気分を変えるために全粒粉の食パンを焼こうと思い、どうせならと全粒粉100%で焼いてみました。
ちゃんと焼けるかなーと心配でしたが、しっかり膨らんで見た目はいい感じ。食感は少しぼそぼそした感じがありますが、サンドイッチ用って思えばいいかも？

来週はこれでいってみます。
